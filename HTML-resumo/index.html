Inspetor de Elementos

Umas das ferramentas mais úteis que temos, principalmente para lidar com erros, é o inspetor de elementos no navegador, com ele é possível literalmente inspecionar os elementos do HTML em seu navegador e entender como ele está se comportando na página. Com inspetor de elementos você tem todas as informações sobre cada tag do HTML.

Estrutura básica do HTML

Dentro do HTML temos uma estrutura básica de elementos que é seguida

A estrutura é composta por:

  • head, que é a tag composta pelos cabeçalhos do documento da página, elementos que não são exibidos dentro do site (como o título da aba);
  • body é a tag onde vai todo o conteúdo da página, tudo nesta tag é o que será apresentado ao visitande do site;
  • header, que é a tag composta pelo cabeçalho dos elementos da página, onde geralmente fica o menu com o direcionamento para o conteúdo do site;
  • main, este nome em português significa "principal" e esta tag é justamente onde colocamos o conteúdo principal do site!;
  • aside é a tag onde irá o conteúdo separado do conteúdo principal mas que tem uma ligação com o mesmo;
  • footer é a parte final do site, onde tem as últimas informações e onde geralmente colocamos detalhes que não estão de fato relacinoado ao conteúdo.

Os atributos são informações que declaramos no momentos em que iniciamos uma tag HTML, nesse atributo passamos informações complementares à tag, como o endereço de uma imagem em uma tag img

Listas ordenadas e não ordenadas

As listas são meios de colocarmos uma sequência de informações relacionadas, literalmente listar elementos. Para isso podemos criar uma lista ordenada (onde a posição de cada elemento importa) com a tag ol, ou a lista não ordenada (onde a ordem dos elementos não importa) com a tag ul
